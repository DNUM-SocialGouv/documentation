# Anonymisation des données

Comme l'indique la [CNIL](https://www.cnil.fr/fr/technologies/lanonymisation-de-donnees-personnelles), l’anonymisation
rend impossible l’identification d’une personne à partir d’un jeu de données et permet, ainsi, de respecter sa vie
privée.

## Checklist

* [ ] Si nécessaire, utiliser un export (dump) anonymisé de la base de données sur les environnements d'intégration et 
  de pré-production.
